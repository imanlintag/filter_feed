# Filter_Feed
Python RSS Filter Feed

This Program creates an RSS Feed for given URLS and Keywords and displays them onto the shell

How to Run: 

Install Python 3 

Import Feed Parser

On your terminal/shell:

Type "python3 filter_feed.py"
